# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text — covering expected behavior, prohibited behavior, enforcement guidelines, and the four-tier enforcement ladder — lives at the link above. Translations and prior versions are linked from the same page.

By participating in this project (opening issues, submitting PRs, commenting, reviewing) you agree to abide by the Covenant.

## Reporting

If you experience or witness behavior that violates the Covenant, report it to the maintainer at **blevene@lightforgeventures.com**. Reports are handled privately. Include as much context as you can — links to the issue / PR / comment, dates, what happened, and any prior interactions that are relevant.

You can also report Covenant violations to GitHub directly via [their abuse reporting flow](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam) if the conduct involves account-level behavior outside the scope of this repository.

## Enforcement

The maintainer is responsible for enforcement. The Covenant's enforcement guidelines (the [four-tier ladder: correction → warning → temporary ban → permanent ban](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)) describe how decisions are made and what consequences different categories of behavior carry.

If a maintainer needs to act on a report involving themselves, decisions are deferred to another active contributor or, failing that, paused pending external mediation.
